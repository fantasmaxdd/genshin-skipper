# 🚀 Genshin Impact Dialogue Skipper

Удобный, быстрый и безопасный фоновый скрипт для автоматического пропуска диалогов и автовыбора реплик в Genshin Impact на ПК. Проект построен на базе компьютерного зрения (OpenCV), имитирует реальные действия пользователя и не внедряется в процессы игры, что делает его на 100% безопасным от блокировок.

## ✨ Особенности проекта
- **🤖 Автоматический пропуск диалогов:** Непрерывно сканирует экран и отправляет нажатие `Space`, как только появляется интерфейс обычной беседы.
- **🎯 Умный выбор реплик:** Находит значок облачка диалога, плавно перемещает курсор мыши на плашку ответа и делает клик (используется небольшой случайный разброс координат клика, чтобы имитировать руку человека).
- **👁️ Защита от яркого фона (Grayscale):** Поиск шаблонов происходит в черно-белом режиме. Скрипт стабильно видит кнопки интерфейса даже на ослепительно белом небе или ярких вспышках.
- **🛡️ Защита от Alt+Tab:** Скрипт автоматически отслеживает активное окно Windows. Если вы свернули игру, открыли браузер или Дискорд, бот мгновенно «засыпает» и не дергает ваш курсор.
- **📦 Работа в системном трее:** Никаких надоедливых черных окон консоли. Программа сворачивается к часам Windows, отображает текущий статус и управляется горячими клавишами.

## 🎮 Инструкция по запуску и использованию

1. Скачайте актуальный файл `GenshinSkipper_0.1.exe` в разделе **Releases** по кнопке выше.
2. Запустите игру Genshin Impact (рекомендуется полноэкранный режим или окно без рамки).
3. Нажмите по скачанному файлу правой кнопкой мыши и выберите **"Запуск от имени администратора"** *(это обязательное требование Windows, иначе система заблокирует эмуляцию кликов поверх игры)*.
4. В системном трее Windows (возле часов) появится иконка программы.

### ⌨️ Горячие клавиши управления:
* **`F8`** — **Включить / Поставить на паузу**. При переключении Windows покажет всплывающее уведомление с текущим статусом бота.
* **Полное закрытие программы:** Кликните правой кнопкой мыши по иконке у часов -> нажмите **Выход**.

## 🛠️ Инструкция для разработчиков

Если вы хотите запустить проект из исходного кода или доработать его под себя, следуйте этим шагам:

### 1. Подготовка окружения
- Склонируйте репозиторий и перейдите в корневую папку проекта.
- Убедитесь, что у вас установлен Python версии 3.10 или новее.
- Активируйте ваше виртуальное окружение (`venv`).

### 2. Установка зависимостей
Установите все необходимые библиотеки одной командой:
```bash
pip install -r requirements.txt
```

### 3. Запуск скрипта
Запустите главный файл проекта обязательно от имени Администратора:
```bash
python main.py
```

## 📦 Сборка собственного автономного .exe

​Для компиляции проекта со всеми вшитыми ресурсами, иконками и конфигами используется утилита PyInstaller. Запустите следующую команду в терминале:
```bash
pyinstaller --clean --onefile --noconsole --add-data "templates;templates" --add-data "config;config" --icon="templates/icon.png" --name "GenshinSkipper" main.py
```

## 📝 Структура проекта

​Содержимое репозитория полностью соответствует архитектуре приложения:
```text
genshin-skipper/
├── config/
│   └── settings.py          # Конфигурация задержек и порога точности OpenCV
├── src/
│   ├── skipper.py           # Логика эмуляции мыши/клавиатуры и проверка Alt+Tab
│   ├── vision.py            # Захват экрана и Grayscale-поиск шаблонов через OpenCV
│   └── input_handler.py     # Обработка нажатий и горячих клавиш
├── templates/
│   ├── dialog_icon.png      # Отрезанный элемент кнопок лога для обычного диалога
│   ├── option_icon.png      # Шаблон белого облачка с тремя точками для выбора реплик
│   └── icon.png             # Иконка приложения для системного трея Windows
├── main.py                  # Точка входа: инициализация фоновых потоков и трея
├── README.md                # Документация проекта (этот файл)
└── requirements.txt         # Список внешних Python-библиотек
```
Разработано с использованием OpenCV, PyDirectInput, PyGetWindow & PyStray.
